### 3.2 颜色规范（详细）

```
主色调：
  - #37352F - 主文字颜色（标题、正文、强调文字）
  - #787774 - 次要文字颜色（副标题、辅助文字、图标）
  - #9B9A97 - 占位符文字颜色（输入框占位符、禁用状态）

背景色：
  - #FFFFFF - 主背景、卡片背景、侧边栏背景
  - #F7F6F3 - 侧边栏背景（可选，如果需要区分）
  - #FAFAF9 - 输入框背景（本页面不涉及）

边框色：
  - #E8E7E5 - 默认边框（卡片边框、分割线）
  - #DDDBD8 - 悬停边框（卡片悬停时）

强调色：
  - #2383E2 - 链接、按钮（本页面不涉及，但可用于未来扩展）

交互状态色：
  - rgba(0, 0, 0, 0.03) - 悬停背景（导航项、FAQ 项）
  - rgba(0, 0, 0, 0.05) - 激活背景（导航项激活状态）
  - rgba(0, 0, 0, 0.08) - 卡片阴影（卡片悬停时）
```

## 7. 设计注释

### 7.1 设计要点

- **信息层次清晰**：使用卡片分组，每个卡片一个主题，便于用户快速定位信息
- **视觉引导**：使用 SVG 图标增强可读性，帮助用户快速识别内容类型
- **内容可扫描**：使用列表和短段落，便于用户快速浏览和理解
- **一致性**：与"我的故事"和"我的PS"页面保持相同的设计语言和交互模式
- **留白充足**：卡片之间保持足够的间距，避免视觉拥挤

### 7.2 参考风格

- Notion 风格的知识库页面：简洁、清晰、易读
- 文档页面设计：信息层次分明，便于查找
- 帮助中心页面：常见问题以问答形式呈现

### 7.3 特别注意

- **内容卡片之间保持足够的间距（32px）**，避免视觉拥挤
- **FAQ 部分可以考虑使用手风琴组件**，节省空间，提升交互体验
- **图标使用 SVG 图标**（线性图标，1.5px 描边，圆角端点），保持风格统一
- **确保文字对比度符合可访问性标准（WCAG AA）**
- **响应式设计要确保移动端体验良好**，内容可读性不受影响
- **页脚链接要有明确的视觉反馈**，让用户知道可以点击
