【方法一：Netlify Drop - 最简单，推荐！】
─────────────────────────────────────────
1. 访问：https://app.netlify.com/drop
2. 将整个文件夹拖拽到页面中
3. 等待上传完成
4. 复制生成的链接（例如：https://xxx.netlify.app）
5. 分享链接即可！

✅ 优点：无需注册、完全免费、自动 HTTPS

─────────────────────────────────────────

【方法二：GitHub Pages】
─────────────────────────────────────────
1. 在 GitHub 创建新仓库
2. 上传 HTML 文件
3. Settings → Pages → 选择 main 分支
4. 访问：https://你的用户名.github.io/仓库名/使用说明页面.html

─────────────────────────────────────────

【方法三：Vercel】
─────────────────────────────────────────
1. 访问：https://vercel.com
2. 使用 GitHub 登录
3. 导入文件夹
4. 点击 Deploy
5. 获得分享链接

─────────────────────────────────────────

【推荐方案】
─────────────────────────────────────────
最快最简单：使用 Netlify Drop
→ https://app.netlify.com/drop

只需拖拽文件夹，即可获得可分享的链接！
