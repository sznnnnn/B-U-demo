# HTML 页面部署指南

本指南提供了几种将 HTML 文件转换为可分享链接的方法。

## 方法一：Netlify Drop（最简单，推荐）

### 步骤：
1. 访问 [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. 直接将包含 HTML 文件的文件夹拖拽到页面中
3. 等待上传完成
4. 系统会自动生成一个链接，例如：`https://random-name-123.netlify.app`
5. 复制链接即可分享

### 优点：
- ✅ 无需注册（但注册后可以管理）
- ✅ 完全免费
- ✅ 自动 HTTPS
- ✅ 支持自定义域名
- ✅ 拖拽即可完成

---

## 方法二：GitHub Pages（适合有 GitHub 账号）

### 步骤：
1. 在 GitHub 上创建一个新仓库
2. 将 HTML 文件上传到仓库
3. 进入仓库 Settings → Pages
4. 选择 Source 为 `main` 分支
5. 点击 Save
6. 等待几分钟，访问 `https://你的用户名.github.io/仓库名/使用说明页面.html`

### 优点：
- ✅ 完全免费
- ✅ 版本控制
- ✅ 可以管理多个文件
- ✅ 支持自定义域名

---

## 方法三：Vercel（快速部署）

### 步骤：
1. 访问 [https://vercel.com](https://vercel.com)
2. 使用 GitHub 账号登录
3. 点击 "New Project"
4. 导入包含 HTML 文件的文件夹
5. 点击 Deploy
6. 获得分享链接

### 优点：
- ✅ 部署速度快
- ✅ 自动 HTTPS
- ✅ 支持自定义域名
- ✅ 免费套餐充足

---

## 方法四：Surge.sh（命令行工具）

### 步骤：
1. 安装 Node.js
2. 安装 Surge：`npm install -g surge`
3. 在包含 HTML 文件的文件夹中运行：`surge`
4. 按提示输入邮箱和密码（首次使用需要注册）
5. 输入域名（或使用随机生成的）
6. 获得分享链接

### 优点：
- ✅ 命令行操作，适合开发者
- ✅ 完全免费
- ✅ 快速部署

---

## 方法五：在线 HTML 预览服务

### 选项：
- **CodePen**: [https://codepen.io](https://codepen.io) - 适合单个 HTML 文件
- **JSFiddle**: [https://jsfiddle.net](https://jsfiddle.net) - 适合测试和分享
- **HTML Preview**: [https://htmlpreview.github.io](https://htmlpreview.github.io) - 直接预览 GitHub 文件

### 使用 HTML Preview：
1. 将文件上传到 GitHub
2. 访问：`https://htmlpreview.github.io/?https://github.com/用户名/仓库名/raw/main/使用说明页面.html`

---

## 推荐方案

**最快最简单**：使用 **Netlify Drop**，拖拽即可完成，无需注册。

**需要版本管理**：使用 **GitHub Pages**，可以管理多个版本。

**需要自定义域名**：使用 **Vercel** 或 **Netlify**，都支持免费自定义域名。

---

## 注意事项

1. **文件路径**：确保所有相关文件（如"我的故事页面.html"）都在同一文件夹中
2. **相对路径**：HTML 中的链接使用相对路径（如 `我的故事页面.html`）
3. **文件名**：避免使用中文文件名，建议使用英文（如 `index.html`）
4. **HTTPS**：所有推荐的平台都自动提供 HTTPS 支持

---

## 快速开始（Netlify Drop）

1. 打开 [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. 将整个 `原型` 文件夹拖拽到页面
3. 等待上传完成
4. 复制生成的链接
5. 分享链接即可！
