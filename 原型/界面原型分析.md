# BüddyUp 界面原型分析

## 📋 现有界面分析

### 界面1：我的故事 - 有内容状态

**页面结构：**
- **左侧边栏（240px）**：固定导航栏
  - Logo: "BüddyUp"
  - 导航菜单：我的故事（激活）、我的PS、使用说明
- **主内容区**：左右分栏布局
  - **左侧卡片（约400px）**：个人信息表单
  - **右侧面板（约600px）**：内容展示区，包含标签页切换

**设计特点：**
- 使用标签页（Tabs）组织内容：个人特质 / 背景经历
- 信息层次清晰：个人简介 → 学术成就 → 优势标签
- 左侧表单 + 右侧预览的经典布局
- 底部固定：反馈入口 + 社交媒体链接

---

### 界面2：我的PS - 有内容状态

**页面结构：**
- **左侧边栏**：同上
- **主内容区**：左右分栏布局
  - **左侧（约500px）**：表单输入区
    - 基础信息：头像、目标国家/院校/专业
    - 项目经历：可添加多个项目卡片
  - **右侧（约500px）**：PS 预览区
    - 顶部工具栏：复制、导出、版本
    - 内容展示区：生成的 PS 文本

**设计特点：**
- 实时预览模式：左侧输入，右侧生成
- 项目经历支持折叠/展开（隐藏的项目经历）
- 大按钮 CTA："生成 PS 草稿"
- 空状态提示："填写你的信息，然后点击'生成 PS 草稿'按钮开始"

---

### 界面3：使用说明

**页面结构：**
- **左侧边栏**：同上
  - Logo: "BüddyUp"
  - 导航菜单：我的故事、我的PS、使用说明（激活）
- **主内容区**：单列布局
  - **页面标题区域**：标题 + 副标题
  - **内容卡片**：多个卡片垂直排列
    - 快速开始卡片
    - 功能特点卡片
    - 常见问题卡片

**设计特点：**
- 单列卡片式布局，信息层次清晰
- 使用图标增强可读性（📖 快速开始、💡 功能特点、❓ 常见问题）
- 内容以列表和问答形式组织，便于快速阅读
- 卡片式设计，每个卡片一个主题

## 🎨 设计模式总结

### 1. 布局模式
- **固定左侧边栏**：240px 宽度，白色背景
- **主内容区**：最大宽度 1200px，左右分栏布局
- **响应式**：大屏左右分栏，小屏上下堆叠

### 2. 导航系统
- **Logo**：顶部居中
- **导航项**：垂直排列，激活状态有背景高亮
- **菜单项高度**：28px
- **菜单项间距**：2px

### 3. 表单设计
- **输入框**：32px 高度，圆角 4px，浅灰背景 (#FAFAF9)
- **按钮**：深灰色主按钮，32px 高度
- **标签**：左侧标签，14px 字号

### 4. 内容组织
- **标签页**：用于切换不同内容视图
- **卡片**：白色背景，圆角 6px，内边距 16px
- **标签（Tags）**：圆角 pill 形状，用于展示优势/特质

### 5. 颜色系统
- **主背景**：#FFFFFF
- **侧边栏背景**：#FFFFFF
- **输入框背景**：#FAFAF9
- **按钮背景**：深灰色（约 #37352F）
- **文字主色**：#37352F
- **文字次色**：#787774

### 6. 交互模式
- **悬停反馈**：菜单项有背景色变化
- **激活状态**：选中项有背景高亮和字重变化
- **按钮状态**：主按钮深色，悬停时加深

## 📐 尺寸规范

- 输入框高度：32px

- 按钮高度：32px
- 按钮内边距：8px 12px
- 按钮圆角：4px

- 副标题：14px，字重 400，颜色 #787774

## 🔄 交互流程

### 我的故事页面
1. 用户填写左侧个人信息表单
2. 右侧实时展示个人特质和背景经历
3. 点击"开始创作"按钮 → 进入编辑模式

### 我的PS页面
1. 用户填写基础信息和项目经历
2. 点击"生成 PS 草稿"按钮
3. 右侧生成 PS 预览
4. 用户可以复制、导出或查看版本历史

### 使用说明页面
1. 用户进入页面查看产品使用指南
2. 浏览快速开始步骤了解基本流程
3. 查看功能特点了解产品能力
4. 查看常见问题解答疑问

---

## 🎯 设计原则

1. **一致性**：所有页面共享相同的侧边栏和布局结构
2. **清晰性**：信息层次分明，左侧输入，右侧预览
3. **引导性**：空状态有明确的操作提示
4. **简洁性**：使用中性色，避免过度装饰
5. **可访问性**：足够的对比度和清晰的焦点状态
