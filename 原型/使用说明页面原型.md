## 3. 详细规格说明

### 3.1 布局结构

#### 左侧边栏（Sidebar）
```
├─ 位置：页面左侧固定
├─ 尺寸：240px × 100vh
├─ 背景：#FFFFFF
├─ 内边距：16px
└─ 包含元素：
   ├─ Logo 区域（高度 60px）
   ├─ 导航菜单（垂直排列）
   │  ├─ 我的故事（未激活）
   │  ├─ 我的PS（未激活）
   │  └─ 使用说明（激活状态）
   └─ 底部留白
```

#### 主内容区（Main Content）
```
├─ 位置：侧边栏右侧
├─ 尺寸：最大宽度 1200px，自适应高度
├─ 背景：#FFFFFF
├─ 内边距：40px 96px（顶部 40px，左右 96px）
└─ 包含元素：
   ├─ 页面标题区域
   ├─ 内容卡片（多个，垂直排列）
   └─ 卡片间距：32px
```

#### 页面标题区域
```
├─ 位置：主内容区顶部
├─ 尺寸：100% × 80px
├─ 背景：transparent
├─ 下边距：32px
└─ 包含元素：
   ├─ 主标题："使用说明"
   └─ 副标题："快速了解如何使用 BüddyUp 创作你的个人陈述"
```

#### 内容卡片（Content Card）
```
├─ 位置：主内容区内，垂直排列
├─ 尺寸：100% × auto（最小高度 200px）
├─ 背景：#FFFFFF
├─ 边框：1px solid #E8E7E5
├─ 圆角：6px
├─ 内边距：24px
├─ 阴影：无（默认），hover 时 0 4px 12px rgba(0, 0, 0, 0.08)
└─ 包含元素：
   ├─ 卡片标题（带图标）
   ├─ 内容列表/段落
   └─ 分隔线（如需要）
```

#### 页脚（Footer）
```
├─ 位置：页面底部固定
├─ 尺寸：100% × 60px
├─ 背景：#FFFFFF
├─ 内边距：16px 96px
├─ 边框：1px solid #E8E7E5（顶部）
└─ 包含元素：
   ├─ 反馈入口（链接）
   └─ 小红书（链接）
```

## 4. 交互说明

### 导航菜单
```
交互点：侧边栏导航项
触发方式：点击
交互效果：
  - 点击"使用说明"项，当前页面高亮显示
  - 点击其他导航项，跳转到对应页面

```

### 内容卡片
```
交互点：内容卡片
触发方式：悬停
交互效果：
  - 鼠标悬停时，卡片阴影出现（0 4px 12px rgba(0, 0, 0, 0.08)）

```

```
交互点：FAQ 项

  - 展开时显示完整答案内容
  - 收起时只显示问题
动画效果：height transition 0.3s ease
```

### 页脚链接
```
交互点：反馈入口、小红书链接
触发方式：点击
交互效果：
  - 点击链接，打开新页面或弹窗
  - 悬停时文字颜色变深，显示下划线
动画效果：color transition 0.2s ease
```

## 5. 响应式设计

```
断点：
- 桌面端：≥1200px
  - 侧边栏固定 240px
  - 主内容区最大宽度 1200px，左右边距 96px
  - 内容卡片单列显示

- 平板端：768px-1199px
  - 侧边栏可折叠或固定
  - 主内容区左右边距 48px
  - 内容卡片单列显示

- 移动端：<768px
  - 侧边栏隐藏，通过汉堡菜单触发
  - 主内容区左右边距 24px
  - 内容卡片单列显示，内边距调整为 16px

适配要点：
- 标题字号在小屏时调整为 20px
- 卡片内边距在小屏时调整为 16px
- 页脚链接在小屏时垂直排列
```

## 6. 设计注释

### 设计要点：
- **信息层次清晰**：使用卡片分组，每个卡片一个主题
- **视觉引导**：使用图标增强可读性（📖 快速开始、💡 功能特点、❓ 常见问题）
- **内容可扫描**：使用列表和短段落，便于快速阅读
- **一致性**：与"我的故事"和"我的PS"页面保持相同的设计语言

### 参考风格：
- Notion 风格的知识库页面
- 简洁的文档页面设计
- 清晰的层次结构和留白

### 特别注意：
- 内容卡片之间保持足够的间距（32px），避免视觉拥挤
- FAQ 部分可以考虑使用手风琴组件，节省空间
- 图标使用 emoji 或 SVG 图标，保持风格统一
- 确保文字对比度符合可访问性标准（WCAG AA）

## 8. 内容示例

### 快速开始卡片内容：
```
1. 填写"我的故事"页面
   详细描述你的个人背景、学术成就和优势特质。信息越详细，AI 越能理解你的特点。

2. 在"我的PS"页面填写目标信息
   选择目标国家、院校和专业。这些信息将帮助 AI 生成更符合目标院校要求的 PS。

3. 添加项目经历
   描述你的项目经验、担任的角色和取得的成果。可以添加多个项目经历。

4. 生成 PS 草稿
   点击"生成 PS 草稿"按钮，AI 将基于你的故事和目标信息生成个性化的个人陈述。
```

### 功能特点卡片内容：
```
• AI 智能生成：基于你的故事自动生成个性化 PS
• 多版本管理：保存多个版本，随时对比和选择
• 一键导出：支持 Word、PDF 等多种格式
• 实时预览：左侧编辑，右侧实时查看效果
• 智能优化：根据目标院校特点调整 PS 风格
```

### 常见问题卡片内容：
```
Q: 如何让生成的 PS 更符合我的风格？
A: 在"我的故事"页面填写越详细，AI 越能理解你的个人特点和写作风格。建议填写个人简介、学术成就和优势特质。

Q: 可以生成多个版本的 PS 吗？
A: 可以，每次生成都会保存为新版本，你可以在版本历史中查看、对比和选择最满意的版本。

Q: 生成的 PS 可以直接使用吗？
A: 生成的 PS 是草稿版本，建议你根据实际情况进行修改和优化，确保内容真实准确。

Q: 支持哪些目标国家？
A: 目前支持美国、英国、加拿大、澳大利亚等主要留学目的地，更多国家正在添加中。
```
